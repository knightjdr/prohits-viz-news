The alpha test for the 2.0 version of ProHits-viz has begun. It can be accessed at: prohits-viz.lunenfeld.ca/v2.

We are currently testing only the dot plot tool with its associated interactive viewer. Some features from the previous version of the interactive viewer have not yet been implemented, including domain and network analysis, as well as image archiving.

To make our tools more approachable for those outside the interaction proteomics community, all bait and prey nomenclature has (or will be) converted to "condition" and "readout". This will affect file naming, so what would previously have been called bait-bait.svg will now be condition-condition.svg.

New features include the ability to save interactive sessions to a file or the browser, navigate large images using the new "minimap" feature, sort conditions by readout abundance either by condition or for one condition relative to another (fold-change), as well as several new ways of selecting conditions/readouts for analysis and image customization.

If you would like to provide feedback, report bugs or request new features, please use GitHub.

Using GitHub
  1. Create an account on GitHub: github.com
  2. Skip the tutorial (unless you plan on coding)
  3. Go to https://github.com/knightjdr/prohits-viz-client/issues
  4. Click on the green "New issue" button to create a new issue and describe the issue, bug or feature.
  5. Alternatively, browse and comment on issues created by others
